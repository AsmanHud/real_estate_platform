# Codex Instructions

This is a fast MVP assignment.

Do:
- Make small, reviewable changes.
- Preserve existing Python ingestion pipeline.
- Keep backend simple: Express + mysql2.
- Keep frontend simple: React + TypeScript + Vite.
- Use existing API response shape unless explicitly changing it.
- After changes, explain what files changed and how to test.

Do not:
- Rewrite the whole project.
- Replace MySQL.
- Add authentication.
- Add maps.
- Add complex state management.
- Modify archived raw data.
